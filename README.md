# preloved-dist (public CDN mirror)

Compiled, public-facing bundle for the Gammill pre-loved machines page, served via
jsDelivr as the **primary** source (the droplet at buildamachine.gammilldealer.com/preloved/
is the fallback). Generated from the private repo `shudocan/preloved-page` by its
`build_embed.py` — do not edit here by hand.

Loader (on gammill.com): CDN primary, droplet fallback. See the private repo's README.
